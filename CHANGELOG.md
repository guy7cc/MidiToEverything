## [1.0.0] - 2026-06-26

最初の安定版リリース。MIDI コントローラーを Windows の汎用「左手デバイス」として使うための
機能が一通り揃い、配布物単体でインストール〜常用できる状態になりました。

### Added
- **豊富なアクション**: キー入力 / テキスト入力 / マウス（クリック・相対/絶対移動・ホイール）/
  メディアキー / ウィンドウ操作 / 仮想デスクトップ切替 / 画面の明るさ / OBS 連携 /
  HTTP・OSC 送信 / MIDI 出力 / マクロ / トグル など（[docs/05_ActionExpansion.md](docs/05_ActionExpansion.md)）。
- **ホールド／相対エンコーダ**: 押している間だけ発火するホールド型、ノブ/フェーダーの回した量・
  向きに反応する相対モード（しきい値・デッドゾーン・反転・感度に対応）。
- **アプリ別ルールの自動有効化（合算モデル）**: 前面アプリにマッチしたルールが自動で有効化され、
  基本ルールとマッチした全ルールの割り当てが合算で働きます（[rule-union](docs/01_PRD.md)）。
- **入力モニター & ラーン**: 受信した MIDI と発火アクションをリアルタイム表示、ワンクリックで
  バインディング対象に取り込み。
- **常駐・トレイ**: システムトレイ常駐、閉じる挙動の選択、起動時トレイ起動、多重起動防止。
- **緊急停止ホットキー**: 一発で全出力を即停止（既定 `Ctrl+Alt+Pause`、変更可）。
- **設定ウィンドウ**: 言語 / 外観（テーマ・アクセント・UI 拡大率）/ デバイス / 更新 / 診断 / 管理。
- **自動アップデート**: 新バージョン通知・ワンクリック更新、更新チャンネル・確認間隔の設定。
- **配布**: self-contained 単一 exe、ポータブル ZIP、MSI インストーラー（タグ push で CI 自動発行）。

### Changed
- ドキュメントとアプリの用語を「ルール」に統一し、合算（union）モデルを明文化。

### Fixed
- **設定ファイルの破損耐性**: `config.json` が破損・部分書き込み・未知のスキーマだった場合でも
  起動時にクラッシュせず、破損ファイルを `config.corrupt[.N].json` として退避したうえで
  既定設定で起動するようにしました（ユーザーが %APPDATA% を手動で開いて復旧する必要をなくす）。

### Notes
- 対応 OS: Windows 10 (2004+) / 11（64bit）。.NET ランタイムのインストールは不要。
- 既知の制約（UIPI による昇格アプリへの送信、MIDI ドライバのデバイス占有、アンチチート）は
  [README](README.md#-困ったときは) を参照。
