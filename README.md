# MidiToEverything 🎹→⌨️

**お手持ちの MIDI コントローラーを、Windows の「万能な左手デバイス」に。**

鍵盤・パッド・ノブ・フェーダーを、キーボードショートカットやマウス操作、音量調整、OBS のシーン切り替えなど、PC のあらゆる操作に変換します。しかも **使っているアプリに合わせて割り当てが自動で切り替わる** ので、イラスト制作中はお絵かきのショートカット、配信中は OBS の操作…と、いちいち設定し直す必要はありません。

> 例えば——
> - 🎛 ノブを回して**ブラシサイズ**やタイムラインを操作
> - 🎹 パッドを叩いて **Ctrl+Z（元に戻す）** やコピペ
> - 🔊 フェーダーで **音量**を調整
> - 🎬 OBS の**シーン切り替え**や録画開始
> - ⌨️ よく使う定型文や**マクロ**をワンタッチ

---

## ✨ できること

- **豊富なアクション** — キー入力・テキスト入力・マウス／カーソル／スクロール・メディアキー（再生/停止/音量）・ウィンドウ操作・仮想デスクトップ切り替え・画面の明るさ・OBS 連携・HTTP/OSC 送信・MIDI 出力・マクロ・トグル など
- **ノブ／フェーダーに最適** — 押している間だけ発火する「ホールド」や、回した量・向きに反応する「相対エンコーダ」に対応
- **アプリごとに自動で有効化** — 前面のアプリに対応するルールが自動で有効に。共通の「基本ルール」は常に有効で、マッチした全ルールの割り当てが合算で働きます
- **かんたんマッピング** — 入力モニターで実際に届いた信号と**発火したアクション**を確認しながら、「ラーン（動かして割り当て）」で直感的に設定
- **見やすい UI** — ライト／ダークテーマ・アクセントカラー・UI 拡大率を選べます（即時反映）
- **常駐して快適** — システムトレイに常駐。閉じてもトレイに残す／起動時にトレイで立ち上げる、などを選択可能
- **安全のための緊急停止** — ホットキー一発ですべての出力を即停止（初期値 `Ctrl+Alt+Pause`、変更可）
- **自動アップデート** — 新しいバージョンが出たらお知らせし、ワンクリックで更新

## ⬇️ ダウンロードとインストール

[**📦 最新リリースはこちら**](https://github.com/guy7cc/MidiToEverything/releases/latest) から、お好みの方をどうぞ。

| | こんな方に | 内容 |
|---|---|---|
| **インストーラー（MSI）** | きちんとインストールしたい | ウィザードに従うだけ。スタートメニュー登録・デスクトップショートカット・Windows 起動時の自動実行も選べます（管理者権限が必要） |
| **ポータブル版（ZIP）** | インストールしたくない／持ち運びたい | 展開して `MidiToEverything.exe` を実行するだけ。インストール・管理者権限とも不要 |

- どちらも **.NET ランタイムのインストールは不要**です（単体で動きます）。
- インストーラーは最初に**表示言語**を選べます（日本語 / English / 中文 / Español / Deutsch / Français / 한국어）。
- 対応 OS: **Windows 10 / 11（64bit）**。

## 🚀 はじめかた

1. **MIDI コントローラーを PC に接続**します（USB など）。
2. **アプリを起動**します。デバイスが自動で認識されます（タイトルバーに表示）。
3. 画面の**入力モニター**で、コントローラーを動かしてみてください。届いた信号がリアルタイムに流れます。
4. **割り当てを作成**します。「ルール編集」を開き、
   - 「ラーン」で割り当てたいキーやパッドを実際に動かして選び、
   - 送りたい操作（例: `Ctrl + Z`）を指定して保存。
5. あとは普段どおり使うだけ。前面のアプリに応じて、対応するルールが**自動で有効になります**。

> 💡 まずは「基本ルール」に、どのアプリでも使う操作（元に戻す・コピー・貼り付けなど）を入れておくのがおすすめです。

---

## 🗂 ルール（アプリごとに自動で有効化）

各ルールは正規表現を1つ持ち、前面アプリがそれにマッチすると**そのルールが有効**になります。MidiToEverything は常に**全ルールを評価**し、マッチした（＝有効な）ルールの割り当てを**合算**して動かします（どれか1つに「切り替わる」のではありません）。

- **基本ルール**: 正規表現に関係なく常に有効な土台（どのアプリでも使う操作を入れておくと便利）。
- **アプリ別ルール**: 対象アプリが前面のときだけ有効。複数のルールが同時にマッチすれば、すべての割り当てが一緒に働きます。
- **重複に注意**: 上書きはありません。基本ルールとアプリ別ルールが同じパッドに割り当てていると**両方発火**します。アプリ別に変えたいときは、各ルールの正規表現で重ならないようにスコープを切ってください。
- **手動の有効/無効**: 自動マッチに関係なく、ルールを手動で強制的に ON/OFF できます（MIDI の「ルール切替」アクションで指定ルールをトグルすることも可能）。
- アプリの判別は「プロセス名／ウィンドウタイトル」で行います。編集画面の「候補を追加」で、起動中のアプリから選ぶだけで設定できます（細かい調整も可能）。

## ⚙️ 設定

タイトルバーの **⚙ アイコン**から設定ウィンドウを開けます。タブで整理されています。

- **一般**: 表示言語、Windows 起動時に自動実行、トレイへの最小化、緊急停止ホットキー など
- **外観**: テーマ（ライト／ダーク）、アクセントカラー、UI 拡大率
- **デバイス**: MIDI の自動検出、OBS 連携（接続先）
- **更新**: 自動アップデートの確認、更新チャンネル・確認間隔
- **診断**: ログレベルやクラッシュ時の自動再起動、ログフォルダを開く
- **管理**: 設定のエクスポート／インポート／初期化、設定フォルダを開く

設定や作成したルールは自動保存され、**アップデートしても引き継がれます**。バックアップしたいときは「管理」タブの**エクスポート**が便利です。

## 🆘 困ったときは

- **キー操作が効かないアプリがある** … 管理者権限で動くアプリ（や一部ゲーム）には、Windows の仕様（UIPI）上、本アプリも管理者として起動しないと操作を送れないことがあります。
- **DAW など他ソフトと同時に使えない** … 一部の MIDI ドライバはデバイスを占有します。その場合は同時利用ができないことがあります。
- **アンチチート付きのゲームで効かない** … 入力をブロックするゲームは対象外です。
- **おかしくなったら緊急停止** … `Ctrl+Alt+Pause`（変更可）で全出力を即停止できます。
- **設定の保存場所** … `%APPDATA%\MidiToEverything\`（`config.json` と `logs\`）。フォルダごとコピーすればバックアップになります。
